# Health Gap Project 

A web-based awareness project highlighting healthcare inequality in rural Ghana and other underserved communities.

It focuses on challenges such as limited access to medical care, delayed treatment, and lack of health education, while promoting solutions like community outreach, preventive care, and improved healthcare infrastructure.

Live Site: https://bonsu1kwabena-cloud.github.io/health-gap-project/
